Название тест-кейса: Проверка потока данных

ID: TC-005

Описание: Тестирование программы на обработку потока данных при вводе валидных и невалидных значений в переменные.

Предусловия:
1. Программа запущена в консоли Windows.
2. Пользователь имеет доступ к консоли и может вводить данные.

Шаги выполнения:
1. Ввести валидные данные сразу в несколько переменных.
2. Проверить результат.

3. Ввести невалидные данные сразу в любую из переменных.
4. Проверить результат.

5. Ввести данные в невалидном формате сразу в несколько переменных.
6. Проверить результат.

Тестовый набор - TS-005.json

Ожидаемый результат:
1. При вводе валидных данных сразу в несколько переменных программа должна выдать валидный результат.
2. При вводе невалидных данных сразу в любую из переменных программа должна выдать сообщение о вводе невалидных данных.
3. При вводе данных в невалидном формате сразу в несколько переменных программа должна выдать сообщение об ошибке в вводе данных.

Результат тест-кейса:
1. Шаг 1-2 выполняется успешно.
2. Шаг 3-4 выполняется успешно.
3. Шаг 5-6 выполняется неуспешно. Составлен баг-репорт BR-003.txt
