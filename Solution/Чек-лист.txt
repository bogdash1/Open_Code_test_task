Тестирование функциональности.

Проверить, что:
- при нажатии на файл CALENDAR_1.1.exe корректно запускается приложение;
- при валидных входных выводится валидный ответ;
- при невалидных начальной дате И/ИЛИ конечной дате, появляется сообщение об ошибке;
- при появлении валидного ответа возможен выход из программы; 
- при появлении сообщения об ошибке возможен выход из программы. 


Тестирование пользовательского интерфейса.

Проверить, что:
- все текстовые сообщения написанны корректно, с соблюдением правил орфографии и грамматики;
- в текстовых сообщениях отсутствуют "лишние слова", способные запутать пользователя;
- ярлык приложения соответствует технической документации;
- интерфейс приложения соответствует технической документации.


Тестирование на отказоустойчивость.

Проверить, что:
- при вводе валидных данных сразу в несколько переменных программа выдает валидный результат;
- при вводе невалидных данных сразу в любой из переменных программа выдает невалидный результат.
- при вводе данных в невалидном формате сразу в несколько переменных программа выдает сообщение об ошибке.
